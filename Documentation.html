📋
Project Overview

This project is a usability evaluation of the MedCare Patient Portal — a digital health platform designed to help patients manage appointments, prescriptions, messages, and pharmacy interactions. The evaluation was conducted using the DECIDE framework to ensure structured, user-centered assessment.

Key Details
  • Evaluation Method: Moderated Usability Testing + SUS Questionnaire
  • Framework: DECIDE (Determine, Explore, Choose, Identify, Decide, Evaluate)
  • Participants: 6 BSA students from Partido State University
  • Evaluation Period: Academic Term 2024–2025
🎯
Objectives
  • Assess the ease of navigation and first-glance clarity of the MedCare interface
  • Measure task completion rates, time-on-task, and user satisfaction scores
  • Identify usability pain points across core features (login, shop, messaging, profile)
  • Provide actionable recommendations to improve the overall user experience
🛠️
Tools & Materials
Evaluation Tools
  • Screen recording software for session capture
  • Think-aloud protocol for real-time verbal feedback
  • System Usability Scale (SUS) questionnaire
  • Figma for prototype design and interactive walkthrough
Report Tools
  • HTML / CSS / JavaScript for this evaluation website
  • Google Fonts (Playfair Display & Nunito)
  • YouTube for video demonstration hosting
👥
Team Roles & Responsibilities
Roles
  • Team Leader — Overall coordination, task distribution, and final review
  • Facilitator — Guided participants through test sessions using think-aloud method
  • Note-taker — Recorded observations, errors, and participant comments during sessions
  • Data Analyst — Compiled SUS scores, task times, and success rates
  • Prototype Designer — Created and iterated on the Figma prototype
  • Report Author — Authored the written report and built this website
