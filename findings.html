Reflections Additional Findings Prototype Walkthrough Video Demonstration

Key usability issues identified during testing of the MedCare Patient Portal.

Additional Findings

⚡ Performance & Lagging Issues

Users noted that the system lags most of the time during use, affecting efficiency. The system should be made smoother and more responsive to ensure all functions work well.

📋 Need for Step-by-Step Instructions

Users suggested adding step-by-step instructions so that other people can use the system without any confusion, especially those who are unfamiliar with it.

👴 Tutorial for Older Users

For older people, users recommended adding a tutorial or a step-by-step process on how to use the system so that they will be properly guided through it.

🏃 Overall Efficiency Improvement

Users directly called for an improvement in the system's overall efficiency, indicating that the current performance does not fully meet their expectations.

💊 Prescription Validation Concern

Users raised concern about the prescription feature — specifically, how the system can ensure that a doctor's prescription submitted by a user is valid and legitimate.

🔐 PIN/Password Security for Transactions

Since the system involves financial transactions, users suggested adding a PIN or password verification step to confirm that the actual account owner is the one making the purchase.

ℹ️ More Detailed Product Information

Users felt the system needs to provide more information about products, especially since they are health-related and users need to make informed decisions before purchasing.

🛒 Expand Product Variety

Users suggested expanding the shop beyond medicines to include more detailed and diverse products such as water and personal care kits to better serve user needs.

🌏 Add Local Products to the Shop

Users observed that most products available in the shop are foreign. They recommended adding local products that are common in the area to make the platform more relevant and accessible.

🔧 Minor Malfunctions Affecting Outcome

There are minor malfunctions in the system that might affect the final outcome. Improving and enhancing the system to work more smoothly was identified as a key suggestion.

⚠️ Lagging Causes Functional Failures

The system lagging was identified as the primary area needing improvement. Users believe that if the lag is fixed, the system will be fully functional and perform very well overall.
